# Contributing to Shelter

Contributions are welcome! Please follow these guidelines.

## Getting started

1. Look for open issues or create a new one to discuss your ideas before starting.
2. Fork the repository and create a new branch from `dev`:
   ```bash
   git switch -c feature/your-feature-name
   ```
3. Make your changes and ensure the code follows the project's style.
4. Open a Pull Request **targeting the `dev` branch**.

## Notes

- The codebase currently contains comments in Spanish, but you are welcome to submit PRs and communicate in English.
